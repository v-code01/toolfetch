# Phase 3: Plan

**Skill:** `superpowers:writing-plans`.

**Do:** Turn the approved spec into a TDD task breakdown with file ownership and per-task verification. Each task carries its failing test, its minimal implementation, and its commit message. Write the plan to `ledger/<project>/plan.md`.

**Gate command:** none scriptable here; the gate is the Proxy's plan review.

**Proxy prompt:** "Per the charter, review this plan as Vansh would. Redline any task that ships a mock, a stub, or an unbacked claim. Every task must be verifiable for real on this hardware. Approve only a plan that holds to the ledge bar."

**Done condition:** the Proxy approves the plan and `plan.md` is written.
